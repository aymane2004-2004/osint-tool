# 🔍 OSINT Toolkit

A comprehensive web-based Open Source Intelligence (OSINT) tool for gathering publicly available information about individuals across multiple platforms.

## 🎯 Overview

OSINT Toolkit is a Flask-based web application that automates the process of searching for usernames across various social media and professional platforms. It leverages Google search operators to find relevant profiles and presents them in an organized, easy-to-navigate interface.

## ✨ Features

- **Multi-Platform Search**: Automatically searches across YouTube, Instagram, Facebook, GitHub, LinkedIn, and more
- **Web-Based Interface**: Clean, responsive UI built with Flask
- **Real-Time Results**: Instant search results with direct links to profiles
- **Extensible Architecture**: Easy to add new platforms and search sources
- **No API Keys Required**: Uses web scraping techniques (no rate limits or API costs)
